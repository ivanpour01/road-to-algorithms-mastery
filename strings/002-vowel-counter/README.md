# Vowel Counter

## Problem
Count the total number of vowels in a text and display how many times each vowel appears.

## Features
- Ignores uppercase/lowercase differences
- Handles accented vowels
- Tracks individual vowel frequency
- Calculates total vowel count

## Approach
1. Convert text to lowercase
2. Normalize accented characters
3. Iterate through each character
4. Count vowels using a HashMap

## Time Complexity
O(n)

## Space Complexity
O(n)

## What I learned
How to normalize text, count character frequency, and use HashMap effectively.
