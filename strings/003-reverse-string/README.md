# Reverse String

## Problem
Reverse a given word or text.

## Features
- Reverses any input string
- Preserves all characters
- Uses StringBuilder for efficient concatenation

## Approach
1. Read user input
2. Iterate through the string from the last character to the first
3. Build the reversed string

## Time Complexity
O(n)

## Space Complexity
O(n)

## What I learned
How to traverse strings backwards and build reversed outputs efficiently.
